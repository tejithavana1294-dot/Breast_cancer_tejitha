# Breast Cancer Classification Using Deep Learning

## Introduction

Breast cancer is a major health concern that requires timely and accurate diagnosis for effective treatment. This project explores the use of deep learning to classify breast tumors as either benign or malignant using medical diagnostic data. A feedforward neural network is developed to learn patterns from the dataset and perform binary classification. The project demonstrates how artificial intelligence can assist in healthcare by improving the efficiency and reliability of disease prediction while providing hands-on experience with deep learning techniques.

## Objectives

- Design and implement a deep learning model for breast cancer classification.
- Prepare and preprocess the dataset to improve learning performance.
- Train the neural network using suitable optimization techniques.
- Evaluate the model using performance metrics such as accuracy, precision, recall, and F1-score.
- Understand the role of deep learning in supporting medical diagnosis and decision-making.

## Dataset
 Breast Cancer Wisconsin (Diagnostic) Dataset

    Total Samples: 569
    Features: 30
    Classes:
    Malignant (M)
    Benign (B)
## Technologies Used

- **Python** – Core programming language used for model development.
- **TensorFlow** – Deep learning framework used to train the neural network.
- **Keras** – High-level API for building and configuring the model.
- **NumPy** – Supports numerical computations and matrix operations.
- **Pandas** – Used for loading, organizing, and preprocessing the dataset.
- **Scikit-learn** – Used for feature scaling, train-test splitting, and evaluation metrics.
- **Matplotlib** – Generates graphs to visualize training and validation performance.
- **Jupyter Notebook** – Interactive environment for developing and testing the project.
